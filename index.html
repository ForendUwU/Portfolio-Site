Projects
Arcade Owner Simulator
This is a game where you act as the owner of the arcade. Your task is to place arcade machines, after what visitors will come to your arcade and pay in game currency for playing on the machine. Using in game currency, you can buy new arcade machines and improve your arcade room. This game made with Unity using C#.
Falling Spikes
This project made with Unity engine using C#. This is a game where you control a cube that must dodge falling spikes. During the game, the player can pick up a shield that blocks one spike hit on the cube. The player can also collect coins that can then be spent in the store to buy skins.
Organizer
Another project made in Rad Studio using C++ is an Organizer. Organizer is an application for scheduling tasks. When the program starts, the user is shown what tasks he has to perform today, and he also sees a list of all tasks for today in the application window. The user can view tasks for other days, add new tasks for any dates, delete existing tasks, and also mark whether he completed the task or not.
Hanged Man
My first project is the Hanged Man. This is a game in which the user has to guess the word by letters, if the user didn't guess the letter, then the element of the gallows is drawn. If the user has not guessed the word in a certain number of attempts, he loses. This project made in Rad Studio using C++ language.
